    准备工作

    创建github仓库

    首先，你要有一个github帐号，什么？你还没有github帐号？那你已经错过了很多非常棒的开源软件了~！快去注册吧

    然后，你需要创建一个仓库名为 {你的username}.github.io 的仓库

    安装git

    没有git？快去下载吧

    安装hexo

    这里就不搬运了，去官网看教程吧。

    日常使用

    • 创建新博文hexo new post <title>
    • 生成静态文件hexo g
    • 本地模拟hexo s
    • 部署到远端hexo d
